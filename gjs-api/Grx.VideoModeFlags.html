Grx.VideoModeFlags

Video mode flag bits (in the Grx.VideoModeExt structure)

VideoModeFlags.LINEAR

Uses linear memory mapping

VideoModeFlags.ACCEL

Uses an accelerated video mode

VideoModeFlags.FAST_SVGA8

Uses faster mixed-planar access

VideoModeFlags.MEMORY

Uses memory only (virtual screen)
