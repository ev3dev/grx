grx_image_new ()

GrxImage *
grx_image_new (const guint8 *pixels,
               gint width,
               gint height,
               const GArray *colors);

Builds a pixmap from a two dimensional (width by height) array of characters. The elements in this array are used as indices into the color table specified with the argument colors . (This means that pixmaps created this way can use at most 256 colors.)
