Grx.LineOptions

from gi.repository import Grx

line_options = Grx.LineOptions()

Custom line options structure.

Zero or one dash pattern length means the line is continuous. The dash pattern always begins with a drawn section.

Example, a white line 3 pixels wide (thick) and pattern 6 pixels draw, 4 pixels nodraw: |[<!-- language="C" --> GrxLineOptions my_line_options; ... my_line_options.color = GRX_COLOR_WHITE; my_line_options.width = 3; my_line_options.n_dash_patterns = 2; my_line_options.dash_patterns = "\x06\x04"; ... ]|
