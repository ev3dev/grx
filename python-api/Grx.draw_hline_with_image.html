Grx.draw_hline_with_image

@accepts(int, int, int, int, int, Grx.Image)
@returns(none)
def draw_hline_with_image(xo, yo, x, y, width, image):
    # Python wrapper for grx_draw_hline_with_image()

Draws a single line of an image on the current context starting at x, y. The image can be offset from the line by specifying xo, yo different from x, y. The image is repeated if the width of the line is greater than the width of the image.

xo

the alignment X coordinate

yo

the alignment Y coordinate

x

the top X coordinate

y

the left Y coordinate

width

the width of the line

image

the image
