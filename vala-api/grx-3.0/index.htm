    Structs:

    • Color - 32-bit integer for storing color information.

    • VideoModeExt - Video driver mode descriptor extension structure.

    • FrameMemory - Private data structure used by Frame.
    • Frame - Private data structure used by Context.
    • FramedBoxColors - Struct to hold colors for a framed box.
    • LineOptions - Custom line options structure.
    • Point - Struct that holds coordinates of a point for use in polyline/polygon functions.
    • Bitmap - A mode independent way to specify a fill pattern of two colors.
    • Pixmap - A fill pattern stored in a layout identical to the video RAM for filling using 'bitblt'-s.
    • LinePattern - Draw pattern for line drawings.
    • KeyEvent - Structure that holds information about a keyboard key event.
    • MotionEvent - Structure that holds information about a pointer motion event.
    • ButtonEvent - Structure that holds information about a pointer button press event.
    • TouchEvent - Structure that holds information about a touch event.
