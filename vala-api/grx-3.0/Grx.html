Grx

Description:

The GRX3 graphics library.

Content:

Namespaces:

  • ColorInfo - Functions for manipulating the color table.
  • Mouse - Functions for manipulating the mouse cursor.
  • User - Functions for drawing using user-defined coordinate systems.

Classes:

  • Context - Private data structure used by Context.
  • Font - Data structure that represents a font.
  • TextOptions - Data structure that contains text drawing options.
  • Pixmap - A fill pattern stored in a layout identical to the video RAM for filling using 'bitblt'-s.
  • Cursor - Structure to hold cursor state.
  • Event - A union of all event types.
  • Device - Base class for objects that represents in input device.
  • DeviceManager - Base class for device managers.
  • Application - Class for managing the application lifecycle.

Structs:

  • Color - 32-bit integer for storing color information.
  • ColorCell - A mutable color value.
  • VideoDriver - The video driver descriptor structure.
  • VideoMode - Video driver mode descriptor structure.
  • VideoModeExt - Video driver mode descriptor extension structure.
  • FrameDriver - The frame driver descriptor structure.
  • FrameMemory - Private data structure used by Frame.
  • Frame - Private data structure used by Context.
  • FramedBoxColors - Struct to hold colors for a framed box.
  • LineOptions - Custom line options structure.
  • Point - Struct that holds coordinates of a point for use in polyline/polygon functions.
  • KeyEvent - Structure that holds information about a keyboard key event.
  • MotionEvent - Structure that holds information about a pointer motion event.
  • ButtonEvent - Structure that holds information about a pointer button press event.
  • TouchEvent - Structure that holds information about a touch event.

Enums:

  • ColorPaletteType - Indicates the type of color palette that is in use.
  • EgaColorIndex - Indexes for use with the array returned by _get_ega_colors.
  • GraphicsMode - available video modes (for set_mode)
  • FrameMode - Available frame modes (video memory layouts)
  • VideoDriverFlags - Flags used by VideoDriver.
  • VideoModeFlags - Video mode flag bits (in the VideoModeExt structure)
  • ColorMode - Values added to a Color to affect how the color is used in drawing operations.
  • ArcStyle - Indicates how an arc should be drawn.
  • TextHAlign - Horizontal text alignment
  • TextVAlign - Vertical text alignment
  • FontWeight - Describes the weight of the font.
  • FontSlant - Describes the slant of the font.
  • FontWidth - Describes the slant of the font.
  • PixmapMirrorFlags - Flags used by mirror.
  • CursorMode - Mouse cursor modes.
  • PnmFormat - PNM file formats.
  • ModifierFlags - Keyboard key modifier flags.
  • EventType - Indicates the type of event.
  • Key - Key symbols.

Error domains:

  • Error - Error codes returned by GRX functions.
