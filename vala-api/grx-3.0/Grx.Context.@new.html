@new

Description:

public static Context? @new (int w, int h, uint8*[]? memory = null, out Context? where = null)

Creates a new context in system memory using the memory layout specified by get_screen_core.

memory must contain screen_get_n_planes pointers (usually just one, but could be as many as 4) of size screen_get_plane_size. null may also be passed to memory, in which case the memory will be dynamically allocated.

Likewise, where can be an unused Context (e.g. you may want to do this if you want a stack allocated context) or it can be null to dynamically allocate a new context.

Parameters:

w

the width of the context
h

the height of the context
memory

memory location(s) or null
where

an unused Context struct or null

Returns:

where or a new context if where was null. Returns null on error.
